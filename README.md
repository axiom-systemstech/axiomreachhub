# 📡 AXIOM REACH-HUB

**Consola de Datos Libres para Agentes. Extrae e indexa contenido de fuentes públicas sin API.**

[![Demo](https://img.shields.io/badge/demo-live-00e5ff)](https://axiom-systemstech.github.io/axiomreachhub/)
[![Three.js](https://img.shields.io/badge/Three.js-r184-green)](https://threejs.org/)
[![TensorFlow.js](https://img.shields.io/badge/TensorFlow.js-4.22.0-orange)](https://www.tensorflow.org/js)

## 🎯 ¿Qué hace?

- **Extracción simulada** de contenido de Twitter, Reddit, GitHub, IPTV, Hacker News y ArXiv
- **Visualización 3D** con Three.js (partículas de datos)
- **IA integrada** (TensorFlow.js + Transformers.js para resúmenes)
- **Exportación** a Markdown, JSON y copia al portapapeles
- **Gamificación** con logros y racha diaria
- **Historial local** de búsquedas
- **Compartir** configuración por URL
